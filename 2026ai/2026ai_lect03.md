## 回帰分析の全体像

* 多変量回帰（重回帰）:

正則化回帰
* Ridge：係数を縮小（L2 ペナルティ）
* Lasso：係数をゼロにする（L1 ペナルティ）

**次回の型**：カテゴリの予測（分類）

## 線形回帰と最小二乗法

$$y = wx + b$$

**最小二乗法**：誤差の二乗和を最小化するw，bを求める

$$L = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2 \quad \text{（MSE）}$$

$$w = \frac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sum(x_i-\bar{x})^2}, \qquad b = \bar{y} - w\bar{x}$$

→ scikit-learn の `LinearRegression` が自動計算

---

## 評価指標と scikit-learn

**評価指標**

| 指標 | 式 | 解釈 |
|------|-----|------|
| RMSE | $\sqrt{\frac{1}{n}\sum(y_i-\hat{y}_i)^2}$ | yと同じ単位の誤差 |
| R² | $1 - \frac{\sum(y_i-\hat{y}_i)^2}{\sum(y_i-\bar{y})^2}$ | 0〜1，1が最良 |

**scikit-learnの基本操作（3 行）**

```python
model = LinearRegression()
model.fit(X_train, y_train)      # 学習
y_pred = model.predict(X_test)   # 予測
r2 = r2_score(y_test, y_pred)    # 評価
```

## 多変量回帰：複数の特徴量

$$y = w_1x_1 + w_2x_2 + \cdots + w_px_p + b$$

**例：糖尿病進行度の予測（10特徴量）**

| 特徴量 | 内容 |
|--------|------|
| age | 年齢 |
| bmi | 体格指数 |
| bp | 平均血圧 |
| s1–s6 | 血清値6種 |

scikit-learnでは **コードは単回帰と同一** 
→ X の列数が増えるだけ

## 多重共線性（Multicollinearity）

* **問題**：特徴量どうしが強く相関 → 係数の推定が不安定
* **確認**：相関行列のヒートマップ

```python
corr = pd.DataFrame(X, columns=feature_names).corr()
sns.heatmap(corr, annot=True, cmap='coolwarm', center=0)
```

**影響**
- 個々の係数の解釈が困難になる
- モデルが不安定（データが少し変わると係数が大きく変動）

**対策** → 正則化（Ridge が特に有効）

---

## 特徴量の重要度：係数の大きさで解釈

重回帰の係数 $w_j$ は**その特徴量の目的変数への寄与**を表す

```python
coef_df = pd.DataFrame({'特徴量': feature_names,
                         '係数':   model.coef_})\
            .sort_values('係数', ascending=False)
```

**注意点**
- 特徴量が**標準化されている場合のみ**係数の大きさを比較できる
- 糖尿病データは標準化済み → 直接比較可能

**次回の決定木との違い**
- 回帰の係数：正負あり，線形な影響のみ捉える
- 決定木の feature importance：非線形な影響も捉える

## 過学習（Overfitting）

```
         訓練R²    テストR²
1次多項式  低い       低い     ← 未学習（underfitting）
3次多項式  高い       高い     ← 適切なモデル
9次多項式  完璧       低い     ← 過学習（overfitting）
```

**原因**：モデルがノイズまで記憶する

**対策**：損失関数に係数の大きさに対するペナルティを加える
→ **正則化（Regularization）**

$$L_{\text{正則化}} = \underbrace{\frac{1}{n}\sum(y_i-\hat{y}_i)^2}_{\text{予測誤差}} + \underbrace{\alpha \cdot \text{ペナルティ}(w)}_{\text{複雑さへの罰則}}$$

---

## Ridge 回帰（L2正則化）

$$L_{\text{Ridge}} = \frac{1}{n}\sum(y_i-\hat{y}_i)^2 + \alpha\sum_j w_j^2$$

- $\alpha$：正則化の強さ（大きい → 係数がより小さくなる）
- **係数をゼロに近づけるが，ゼロにはしない**
- 多重共線性に強い（係数を安定させる）
- 特徴量をすべて維持したい場合に向く

```python
from sklearn.linear_model import RidgeCV
model = RidgeCV(alphas=np.logspace(-3, 3, 50), cv=5)
model.fit(X_train, y_train)
print(f"最適α: {model.alpha_:.4f}")
```

---

## Lasso 回帰（L1 正則化）

$$L_{\text{Lasso}} = \frac{1}{n}\sum(y_i-\hat{y}_i)^2 + \alpha\sum_j |w_j|$$

- **係数を完全にゼロにできる** → 自動的な特徴量選択
- 不要な特徴量の係数が正確にゼロになる
- 解釈しやすいスパースなモデルが得られる
- 多重共線性がある場合は不安定になることがある

```python
from sklearn.linear_model import LassoCV
model = LassoCV(alphas=np.logspace(-3, 3, 50), cv=5)
model.fit(X_train, y_train)
print(f"最適α: {model.alpha_:.4f}")
print(f"ゼロ係数: {np.sum(model.coef_ == 0)}/{X.shape[1]}")
```

---

## Ridge vs Lasso：詳細比較

| 観点 | Ridge（L2） | Lasso（L1） |
|------|------------|------------|
| ペナルティ | $\sum w_j^2$ | $\sum\|w_j\|$ |
| 係数の挙動 | ゼロに近づく（完全ゼロはなし） | 完全にゼロになる |
| 特徴量選択 | しない（全特徴量を保持） | する（不要な特徴量を除外） |
| 多重共線性 | 強い（係数が安定） | 弱い（どれかをゼロに） |
| 解釈性 | やや低い | 高い（選択された特徴量のみ） |
| **使い分け** | 全特徴量が関連していそうな時 | 無関係な特徴量が多い時 |

---

## α の選択：交差検証（Cross Validation）

**交差検証（5分割CV）**：データを5分割し，
4分割で学習→1分割で評価を5回繰り返す

```python
┌───┬───┬───┬───┬───┐
│ 1 │ 2 │ 3 │ 4 │ 5 │  ← データ
└───┴───┴───┴───┴───┘
 検証  学習  学習  学習  学習  → 1回目
 学習  検証  学習  学習  学習  → 2回目
       ...
 学習  学習  学習  学習  検証  → 5回目
```

**RidgeCV・LassoCVが自動で最適αを選択**
→ テストデータに触れずに αを決定できる
