## PCA の直観：最大分散方向を探す

**問い**：400 次元を1次元に圧縮するとき，どの方向に射影するか？

<!--

      ×  ×
        ×  ×

-->

**PCA の答え**：**データの分散が最大になる方向**を選ぶ

**なぜ分散が重要か**
- 分散が大きい = データ間の差異が保存される
- 分散が小さい = 情報が失われる（点が一点に集まる）

$$\text{第1主成分} = \underset{\|w\|=1}{\arg\max}\; \text{Var}(Xw)$$

---

## PCA の数学：固有値分解

**共分散行列** $\mathbf{C} = \frac{1}{n} X^\top X$

$\mathbf{C}$ を固有値分解すると：

$$\mathbf{C} = V \Lambda V^\top$$

- $V$：固有ベクトル行列（= 主成分の方向）
- $\Lambda$：固有値の対角行列（= 各主成分の分散）

**固有顔の正体**

$$\text{固有顔} = V \text{ の各列} = \text{顔空間の「基底ベクトル」}$$

- 第1固有顔：最も大きな分散方向（全サンプルに共通する最大の変動）
- 第2固有顔：第1に直交しつつ次に大きな分散方向
- …

→ 第5回で可視化した「固有顔の形」の数学的な意味

---

## PCA と SVD の関係

**特異値分解（Singular Value Decomposition）**

$$X = U \Sigma V^\top$$

- $U$：左特異ベクトル（サンプル空間）
- $\Sigma$：特異値の対角行列
- $V$：右特異ベクトル（**＝ PCA の主成分**）

**対応関係**

| PCA | SVD |
|-----|-----|
| 固有ベクトル $V$ | 右特異ベクトル $V$ |
| 固有値 $\lambda_i$ | $\sigma_i^2 / n$（特異値の二乗） |
| 主成分スコア $XV$ | $U\Sigma$ |

**なぜ SVD が重要か**
- 数値的に安定（共分散行列の計算が不要）
- `sklearn.decomposition.PCA` の内部実装も SVD ベース

---

## PCA の限界と NMF への動機

**PCA の限界**

**1. 線形性**  
- 非線形な構造（多様体）は捉えられない

**2. 負の係数を含む**  
- 固有顔は正の成分と負の成分が混在
- → 「引き算で顔を作る」＝ 直観に反する

**3. 解釈の難しさ**  
- 各固有顔はデータ全体の「幽霊のような」パターン
- 「目だけ」「口だけ」のような局所的な部品ではない

**NMF が解決すること**

$$V \approx W \times H \quad \text{（ただし } W, H \geq 0\text{）}$$

→ **すべての係数が非負** = 「足し合わせる」だけで顔を表現

---

## NMF：非負値行列因子分解

**Non-negative Matrix Factorization（Lee & Seung, 1999）**

**基本的な考え方**

```
顔の画像 ≈ 「目の成分」×0.8 + 「鼻の成分」×0.6 + 「口の成分」×0.4 + …
```

→ 顔を**局所的な部品の加算**で表現

**PCA との比較**

|       | PCA（固有顔） | NMF（局所顔） |
|:------|-------------|-------------|
| 係数   | 正・負どちらも | **非負のみ** |
| 基底の形 | 顔全体のパターン | **局所的な部品** |
| 再構成 | 加算・減算 | **加算のみ** |
| 解釈   | 難しい | **自然（部品ベース）** |

---

## NMF と心理学：部品ベース表現

**Lee & Seung (1999) の主張**

> 人間の視知覚は対象を「部品の集合」として表現している
> NMF はこれを自然に実現する

**心理学的な接点**

- **Wittgenstein の部品知覚**: 複雑な対象は構成要素の組み合わせとして理解
- **Hubel & Wiesel（第7回）**: 視覚野のニューロンは局所的な特徴に反応
- **顔知覚研究**: 顔の部品（目・鼻・口）は個別に処理される

**実習での確認**

- PCA の固有顔：全顔的なパターン
- NMF の基底：局所的な顔部品が現れる（ことが多い）

## k-means クラスタリング

**教師なし学習の典型**：ラベルなしでデータをグループ化

**アルゴリズム**

1. k 個の中心点をランダムに初期化
2. 各サンプルを最近傍の中心点に割り当て
3. 各クラスターの平均で中心点を更新
4. 割り当てが変化しなくなるまで 2-3 を繰り返す


**損失関数**（Within-Cluster Sum of Squares）

$$L = \sum_{j=1}^{k} \sum_{x \in C_j} \|x - \mu_j\|^2$$

→ クラスター内の散らばりを最小化

**本日の設定**

- PCA 空間（100次元）でクラスタリング
- クラスター中心を顔画像として可視化（`pca.inverse_transform`）

---

## k-means の選択：k の決め方

**エルボー法（Elbow Method）**

```
k を増やすと Within-SS（惰性）は必ず減少
↓
急激に減少が鈍くなる点 = 「肘」= 適切な k
```

**シルエット係数（Silhouette Score）**

$$s(i) = \frac{b(i) - a(i)}{\max(a(i), b(i))}$$

- $a(i)$：同クラスター内の平均距離
- $b(i)$：最近傍クラスターへの平均距離
- $s \to 1$：良いクラスタリング

**本データでの期待**

- 40人を k 個のグループに集約
- 性別・年齢・特徴的な顔の傾向でグループが形成されるか？

## SOM：自己組織化マップ（概観）

**Self-Organizing Map（Kohonen, 1982）**

**基本的なアイデア**

```
高次元データを 2D グリッドに「写像」
→ 似たデータは近くに，異なるデータは遠くに配置
→ データの「地図」が作られる
```

**k-means との違い**

|     | k-means | SOM |
|-----|---------|-----|
| 出力 | クラスターラベル | **2次元マップ** |
| トポロジー | 保存しない | **局所構造を保存** |
| 直観 | データのグループ化 | データの「地図作り」 |

**心理学との接続（第7回へつながる）**

- 大脳皮質の**トポグラフィックマップ**（体性感覚野，視覚野）
- 近くの刺激は近くのニューロンが処理する
- → Hubel & Wiesel（第7回）への橋渡し

## Infomax の位置づけ（概観）

**Information Maximization（Bell & Sejnowski, 1995）**

**原理**：出力の情報量（エントロピー）を最大化するように変換を学習

$$\max_W H(y) = \max_W H(f(Wx))$$

**何が得られるか**

- 入力データの**統計的に独立な成分**を抽出
- Independent Component Analysis（ICA）の理論的基礎

**PCA・NMF との関係**

| 手法 | 最適化の目標 | 制約 |
|------|-----------|------|
| PCA | 分散最大化 | 直交性 |
| NMF | 再構成誤差最小化 | 非負性 |
| ICA（Infomax） | 独立性最大化 | 統計的独立 |

**深入りしない理由**：数学的な前提知識が必要。概念のみ押さえる

## TF-IDF と Naive Bayes（10分概説）

**文書の数値表現：TF-IDF**

```
「機械学習」という単語が：
  1 つの文書に多く出現（TF↑）かつ
  多くの文書に出現しない（IDF↑）
  → その文書を特徴づける重要な単語
```

$$\text{TF-IDF}(t, d) = \underbrace{\frac{\text{count}(t, d)}{|d|}}_{\text{TF}} \times \underbrace{\log\frac{N}{|\{d: t \in d\}|}}_{\text{IDF}}$$

**生成モデル：Naive Bayes**

$$P(y \mid x) \propto P(y) \prod_j P(x_j \mid y)$$

- 各特徴量が独立と仮定（「ナイーブ」な仮定）
- **第4回の尤度・最尤推定と直結**：クラス条件付き尤度を最大化
- テキスト分類・スパムフィルタの定番手法
